## 📦 How to Use

### Option 1 — Run Locally
```bash
# Just download and open!
open JARAS_AI.html
```
No installs. No servers. Works right in your browser.

### Option 2 — Deploy on GitHub Pages
1. Fork or upload this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch
4. Your live URL: `https://yourusername.github.io/jaras-ai/JARAS_AI.html`

## 📊 Daily Limit

JARAS has a **20 messages/day** limit to keep API usage in check. The progress bar at the bottom of the chat shows how many you have left. Resets every day at midnight. 🌙

---

## 🔑 API Setup

JARAS uses the **Anthropic Claude API** which is handled by the Claude.ai platform when running inside it. If running standalone, you'll need to add your API key:

```javascript
// In the fetch headers inside JARAS_AI.html
headers: {
  'Content-Type': 'application/json',
  'x-api-key': 'YOUR_API_KEY_HERE',
  'anthropic-version': '2023-06-01'
}
```

Get your API key at [console.anthropic.com](https://console.anthropic.com)
